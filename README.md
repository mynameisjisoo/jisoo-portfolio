### 4. 이 프로젝트를 통해 배운 것

```
🌱 Styled Component를 처음으로 이용해 봤는데, 컴포넌트 단위의 스타일링과  theme을 지정할 수 있는게 편해서 좋았다.
🌱 Swiper 라이브러리 역시 처음 적용해봤는데, 사용법은 어렵지 않았지만 공식 문서를 잘 읽어보는 것이 중요한 것을 알게되었다.
🌱 부모 컴포넌트에서 자식 컴포넌트의 ref를 다룰 때 사용하는 forwardRef hook에 대해 배웠다.
```
